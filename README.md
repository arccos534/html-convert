# Bitrix News Builder (MVP)

Визуальный редактор новостных страниц для старого Bitrix с экспортом в **один автономный HTML-файл**.

## Что реализовано

- Визуальная сборка страницы из блоков (левая библиотека + центральный canvas + правая панель настроек)
- Drag-and-drop перестановка блоков
- Выбор блока, удаление и дублирование
- Rich-text редактирование (TipTap):
  - H1/H2/H3
  - жирный, курсив, подчеркивание
  - цвет текста
  - размер текста
  - выравнивание
  - списки
  - цитаты
  - ссылки
- Набор блоков:
  - hero
  - новостной блок (заголовок/подзаголовок/лид)
  - текстовый блок
  - callout/плашка
  - важно
  - цитата
  - блок с фоном
  - разделитель
  - кнопка
  - таблица
  - колонки
  - карточки
  - изображение
  - статистика
  - диаграмма
- Live preview (рендер экспортного HTML в iframe)
- Работа с изображениями:
  - URL или загрузка файла
  - размер
  - выравнивание
  - подпись
  - скругление
  - тень
  - подложка
- Сохранение проекта:
  - localStorage
  - экспорт JSON-черновика
  - импорт JSON-черновика
- Верхняя панель действий:
  - сохранить
  - открыть
  - preview
  - экспорт
  - копировать HTML
  - скачать HTML

## Главное: экспорт для Bitrix

Экспорт формирует **один `.html` файл**, где:

- все CSS уже встроены в `<style>`
- не нужны отдельные css/js файлы
- нет внешних UI-зависимостей
- HTML открывается как самостоятельная страница
- HTML можно копировать кодом или скачивать файлом

## Запуск

```bash
npm install
npm run dev
```

Продакшен-сборка:

```bash
npm run build
npm run preview
```

## Архитектура (основное)

- `src/types.ts` — доменные типы документа и блоков
- `src/data/blockFactory.ts` — фабрики блоков и библиотека шаблонов
- `src/data/demoArticle.ts` — демо-страница
- `src/components/editor/*` — layout редактора, панели, canvas, top bar
- `src/components/blocks/*` — визуализация и rich-text блоков
- `src/lib/exportHtml.ts` — независимый модуль экспорта в единый HTML
- `src/lib/documentIO.ts` — JSON/файлы/clipboard/localStorage

## Примечание

В продакшен-сборке есть предупреждение о размере JS-бандла (из-за TipTap и dnd). Для MVP это ожидаемо.
